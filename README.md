# 🎓 ClassTrack: Never Miss a Goal.

**Take Control of Your Degree with the Ultimate Student Productivity Platform.**

ClassTrack is a professional-grade attendance management and academic productivity tool designed specifically for school and college students. It helps you navigate the "Attendance Math" of university life by giving you real-time data on exactly how many classes you can safely skip while maintaining your targets.

---

## 🚀 Key Features

- **✅ One-Tap Attendance**: Log your presence in under 2 seconds. No friction, no guilt.
- **📊 Real-time Analytics**: See your current percentage, classes attended, and classes missed at a glance.
- **📉 "Missable" Intelligence**: Automatically calculates exactly how many more classes you can miss while staying above your target (e.g., 75%).
- **📅 Visual Dashboard**: A modern, clean interface that maps out your entire semester.
- **📰 Expert Content Hub**: Integrated blog with 20+ high-quality articles on study hacks, networking, and productivity.
- **🌙 Responsive & Modern**: Built with a "premium" dark-mode aesthetics using Tailwind CSS.

---

## 🛠 Tech Stack

ClassTrack is built using cutting-edge web technologies:

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router, Server Components)
- **Deployment**: [Firebase](https://firebase.google.com/) (Auth, Firestore)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Components**: [Radix UI](https://www.radix-ui.com/) & [Lucide Icons](https://lucide.dev/)
- **State Management**: React Hooks & Context API

## 📦 Getting Started

To run ClassTrack locally:

1. **Clone the repository**:
   ```bash
   git clone [your-repo-url]
   cd attendance-tracker
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up Firebase**:
   Create a `.env.local` file with your Firebase credentials:
   ```env
   NEXT_PUBLIC_FIREBASE_API_KEY=your_key
   NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=your_domain
   ...
   ```

4. **Run the development server**:
   ```bash
   npm run dev
   ```

Open [http://localhost:9002](http://localhost:9002) to view the app.
