# **App Name**: ClassTrack

## Core Features:

- Subject Creation: Allow users to add subjects with name, attendance requirement (%), and teacher name.
- Attendance Marking: Enable users to mark attendance as present or absent for each subject.
- Real-time Calculation: Calculate and update attendance percentage in real-time.
- Attendance Status Indicator: Provide a color-coded status (green, yellow, red) to indicate the attendance status of each subject.
- Classes Missed Calculation: Automatically calculate the number of classes a user can miss or needs to attend to meet the minimum attendance requirement; this tool reasons over attendance data to personalize its result.
- Edit/Delete Subject: Allow users to edit subject details or delete subjects with confirmation.
- Data Persistence: Save attendance data locally to prevent data loss when the app is closed.

## Style Guidelines:

- Primary color: Soft blue (#90AFC5) to convey reliability and calm focus.
- Background color: Light gray (#E5E8E9), nearly desaturated version of the primary, creates a neutral, clean backdrop.
- Accent color: Muted violet (#807593), analogous to the primary but significantly different in brightness, for key actions and highlights.
- Body text: 'PT Sans' (sans-serif) for easy reading of information; Headline: 'Space Grotesk' for emphasis.
- Use simple, clear icons for subject categories and actions (present, absent, edit, delete).
- Dashboard: Use clear sectioning for subjects, employing a card layout for information architecture
- Subtle animations: subtle transitions when marking attendance, like a change in color on button press.
